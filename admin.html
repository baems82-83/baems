Connect

This admin tool talks directly to GitHub and Cloudinary from your browser — nothing is stored on a server in between. Your token is saved only in this browser's local storage. You'll need: 1) a GitHub token with write access to baems82-83/baems, and 2) a Cloudinary unsigned upload preset. See the setup notes below the form if you haven't made these yet.

Create this once in Cloudinary → Settings → Upload → Upload presets → Add upload preset → Signing mode: Unsigned. Copy its name here.

GitHub token: Settings → Developer settings → Personal access tokens → Fine-grained tokens → New token → limit repository access to just baems → Permissions → Contents: Read and write.
