Students

Not shown as a public "directory" page, but this file is fetched directly by the Student Portal login, which means it's served publicly by Netlify no matter whether the GitHub repo is public or private — that part is unavoidable with a backend-free site. Passwords are stored as scrambled hashes (not plain text), so this data isn't fully invisible, but casual viewers can't read passwords directly. Avoid adding anything more sensitive than name, roll number, class/section, and major.

Staff (Teacher) Accounts

Create login accounts for teachers. Teachers log in on this same page (choose "Teacher" at login) and get the Gallery and Resources tabs only — not Site Settings, Notices, Faculty, or Students.
Important: a teacher's save uses the same GitHub token embedded in this file — the tab restriction is a UI convenience, not a hard security wall. Only give teacher logins to people you'd trust with full admin access too.
